# Frequently Asked Questions

## What can I do with this mod?

PIP lets you collect and process biological specimens for virology and pathology research. You can find specimens through exploration, prepare them using lab equipment, and build your knowledge of the biological threats in the post-apocalypse.

## Is this mod complete?

No. PIP is in **early development** (v0.1.0, scaffold phase). The core architecture is in place, but gameplay content is still being built out. Expect new features, specimens, and processing chains in future updates.

## What mods does PIP require?

**PhobosLib** is the only hard dependency. It provides shared utilities used by all Phobos mods.

## What mods does PIP work with?

PIP has optional cross-mod support for:

- **PhobosChemistryPathways** (PCP) -- Chemistry reagents and lab equipment
- **ZScienceSkill** -- Science skill progression from specimen research
- **Expanded Helicopter Events** (EHR) -- Additional specimen sources
- **Dynamic Trading** -- Specimens in the trading economy

All cross-mod features are optional. Installing only PIP (plus PhobosLib) works fine.

## Does PIP conflict with other mods?

PIP is designed to avoid conflicts. It uses its own namespaced items, recipes, and module prefixes (`PIP_`). If you encounter a conflict, please report it.

## Where can I report bugs or suggest features?

Use the [GitHub Issues](https://github.com/phobos-dthorga/mod-pz-industrial-pathology/issues) page or leave a comment on the Steam Workshop listing.
